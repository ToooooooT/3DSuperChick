## keycode:
* 'C' : change code camera
* playing mode:
  * 'W','A','S','D' : move
  * $\uparrow$ $\downarrow$ $\leftarrow$ $\rightarrow$ : move
  * space : jump
* placing mode:
  * 'W','A','S','D' : move
  * left control : move up
  * space : move down
  * 'Z' : get into placing mode
  * 'E' : confirm placing place
  * hold left click : rotate horizontally
  * hold right click : rotate vertically
